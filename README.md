#how to start the project

1- Download this repo to your pc.

2- Open two terminal windows in the said prj directory.

3(a)- for frontened-
cd client
npm install
npm run dev

3(b)- for backened-
cd server
node --env-file=config.env server

Yes, your website is ready for you to store your said records.
